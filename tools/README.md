# Tools

脚本、自动化工具和辅助程序放在这里。

当前常用脚本：

- `generate-app-content.ts`：从 `content/` 生成 `app/content-data.js`。
- `validate-content.ts`：校验结构化内容和素材引用。
- `report-assets.ts`：输出或写入素材体积报告。
- `generate-display-derivatives.ts`：用 ffmpeg 生成 `app/assets/optimized/**/*.webp` 展示派生图。
- `generate-scroll-tiles.ts`：规划或生成长卷 WebP 瓦片。
- `run-e2e.ts`：稳定启动本地或外部地址 E2E。
