# design/

本目录与根目录 `design.md` 直接关联，用于收集后续设计参考资料，并从中归纳适合《西湖繁胜全景图》项目的设计语言。

## 放置内容

- 参考图：Landing、地图、长卷、札记、社区、动效截图等。
- 视觉样张：色彩、纸纹、字体、版式、按钮、标签、浮层、图标等。
- 对标案例：网站截图、局部交互截图、设计稿导出图。
- 归纳笔记：从参考资料中提炼出的可复用规则、禁用项和修改建议。

## 建议命名

- `landing-主题-版本.png`
- `map-主题-版本.png`
- `scroll-主题-版本.png`
- `notes-主题-版本.png`
- `inquiry-主题-版本.png`
- `motion-主题-版本.gif`
- `reference-来源-主题.md`

## 使用规则

1. 参考资料可以先放入本目录，不代表已经成为项目设计规范。
2. 被确认适合本项目的设计语言，应再沉淀回根目录 `design.md`。
3. 原始参考图、截图、导出物是否提交 Git，需要先由用户确认。
4. 归纳时优先提炼：构图、色彩、字体、纹理、组件、动效、禁用项。
5. 与 `design.md` 冲突时，以 `design.md` 中已确认的项目规范为准。
